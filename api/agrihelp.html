Installation:
● User can install the application from google playstore from the below link -
● https://play.google.com/store/apps/details?id=in.co.vnrseeds.www.farmers&hl=en
Login
● Here user has to enter the username and password .
● After entering valid user name and password user has to click the login button.(Make sure user
has a good internet connection)
● After successful login it will redirect the user to the dashboard page.
Navigation
● The navigation page includes different menu buttons i.e.
1. Home
2. Farmer
3. Report
4. Profile
5. Fetch Farmer Records
6. Update Farmer Records
7. Fetch Agreement Records
8. Update Agreement Records
9. Help
10. Change Password
11. Logout
●
While clicking on home menu button it will show the dashboard page.
●
While clicking on the farmer menu button it will show two category i.e
1. Search Farmer
2. Add Farmer
3. Search Agreement
●
While Clicking on the Report menu button it will show us the total records of the farmers including
search option and farmer details.
●
While clicking on the Profile menu button it will show us the user’s details.
●
While clicking on the Fetch Farmer records menu button it will fetch all data from the server and
will show us a dialogue that record fetched successfully.
●
While clicking on the Update Farmer records menu button it will update our local data to server.
●
While clicking on the Fetch Agreement Records menu button it will fetch all data from the server
and will show us a dialogue that record fetched successfully.
●
While clicking on the Update Agreement records menu button it will update our local data to
server.
●
While clicking on the Help menu button it will show us the use of the application.
●
While clicking on the Change password menu button it will redirect us to the change password
page where user can change his/her password.
●
While clicking on Logout menu button it will redirect us to the login page.
Farmer
● While clicking on the farmer menu button it will show two category i.e
1. Search Farmer
2. Add Farmer
3. Search Agreement
Add Farmer
● By clicking on Add farmer button it will redirect us to the add farmer page where user can enter
his
1. Name
2. Contact No
3. Adhar document
4. Passbook document
5. Any ID proof document
6. Any Address proof document
● To the right side of the documents field there is a browse button .By clicking on the browse button
user can select image from his/her gallery or user can capture a picture.
● Passbook Document is mandatory where as other documents are optional.
● By entering all the mandatory fields user has to submit the data by clicking on the submit button.
● The submitted data will save in local database as well as it will go to the server database and it
will create a temporary Farmer ID which will appear in a dialogue.(The user has to make sure that
he/she has a network connection)
● If the user has no internet connection while submitting the data then the data will store in local
database and user can Update the local data by clicking on the Update Records menu button
when user will get internet connection.
Search Farmer
● By clicking on the search farmer button it will redirect us to the search farmer page .
● In this page user can see the total records . And user can search the name/contactno of
the farmer to see the farmer details.
● There is a status column in the farmer details row.If the status is showing tick mark then
the record is from server and it can not be edited and If the status is showing edit mark
then the record is from local database and user can edit the record .
● The user can see the farmer details by scrolling right.
● The user can view the uploaded document by clicking the view doc from the record.
Search Agreement
● By clicking on the search Agreement button it will redirect us to the search agreement
page .
● In this page user can see the total records . And user can search the farmer
name/agreeno/father name/famer id/organiser name/crop of the farmer to see the
details.
● There is a status column in the details row.If the status is showing tick mark then the
record is from server and it can not be edited and If the status is showing edit mark then
the record is from local database and user can edit the record .
● The user can see the farmer details by scrolling right.
● By clicking on the row the details page will apper having 3 categories.
● It is having 3 categories such as
1. Date of soaking and transplanting
2. Crop condition and stage in acrage
3. Harvesting and Dispatch
● Here from the date of soaking and transplanting category the total sum value of Standing Acrage
and Plough down acrage should not be greater than Transplanting acrage.
● By default two fields are mandatory (Soakingdate female,Soakingdate male1)
● The total sum value of Crop condition acrage excellent ,Crop condition acrage Good and Crop
condition acrage Poor should not be greater than Standing Acrage.
● The value of Crop condition acrage Tillering / Kneeheigh and the value of Crop condition acrage
PPI datasseling should not be greater than Standing acrage.
● And if from the harvesting and dispatch category dispatch date field value will be enter then all
the fields from the 3 categories will be mandatory/required.
Profile
● User can see his/her details by clicking on the Profile menu button from the navigation.
● Profile page includes
1. Name
2. Contact
3. Email
4. Status
5. State
Fetch Records
● By clicking on the Fetch Records menu button from the navigation, user can fetch all the records
from server and user can see the details in the search page by searching the name or contact no
of the farmer. ( User has to make sure he/she has a good internet connection )
Update Records
● By clicking on the Update Records menu button from the navigation, User can update all the local
records to the server.( User has to make sure he/she has a good internet connection )
Change Password
● By clicking on the change password menu button from the navigation it will redirect us to the
change password page.
● Here user has to enter his current password.Then he/she has to enter new password and confirm
the new password.Then user has to click the button change password.It will show a dialogue that
password changed successfully and redirect the user to the login page.
Logout
● By clicking on the Logout menu button from the navigation drawer it will logout the user from the
application and redirect the user to the login page.
